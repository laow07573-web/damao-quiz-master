# 呆猫刷题宝

DeepSeek AI 驱动的本地刷题助手 · 支持单选/多选/判断/填空 · Windows + Android 双端

本软件由 B站 **笨蛋鱼坏蛋猫** 开发

##特别说明，本软件只是因为作者受不了市面上的收费会员制的刷题软件，借助deepcode开发的及其低质的小软件，因为专业类别不是计算机，后续维护可能无法实现，还请谅解。

## 功能一览

| 功能 | 说明 |
|------|------|
| 📥 DOCX 导入 | 上传 .docx 文件，AI 自动提取题目、选项、答案 |
| 🧠 AI 解析 | 每道题自动生成考点分析、解题思路、答案详解 |
| 💬 AI 追问 | 对解析内容继续提问，AI 深入讲解 |
| 📊 薄弱分析 | 自动统计各方向正确率，给出针对性学习建议 |
| 📝 错题本 | 手动收藏 + 答错 3 次自动收录，一键重刷 |
| 📋 题型支持 | 单选题、多选题、判断题、填空题 |
| ✅ 多选确认 | 多选模式点击勾选，统一确认提交 |
| 📈 刷题小结 | 每轮结束 AI 自动生成小结 |

## 安装

### Windows

1. 解压 `刷题宝.zip`
2. 双击 `flashcard_app.exe` 运行

### Android

1. 将 `刷题宝.apk` 传到手机
2. 允许安装未知来源应用后安装

---

## 使用指南

### 配置

右上角设置 → 填入 [DeepSeek API Key](https://platform.deepseek.com/api_keys)

### 导入题库

首页「导入题库」→ 选择 `.docx` 文件 → AI 自动解析 → 预览确认

支持题型：单选题、多选题、判断题、填空题

### 开始刷题

勾选题库 → 「开始刷题」→ 作答 → 查看解析 → 追问 → 下一题

- 多选题目：点击选项勾选，底部显示「确认提交 (已选N项)」
- 填空题目：输入文字后点击「提交答案」
- 做错自动展示 AI 解析，做对可手动点击「看解析」

### 错题重刷

首页「错题重刷」→ 自动抽取答错 3 次以上 + 手动收藏的题目，随机洗牌，50 题一轮

### 薄弱分析

首页自动展示，点击「刷新」重新生成
