## 使用指南

### 配置

右上角设置 → 填入 [DeepSeek API Key](https://platform.deepseek.com/api_keys) → 可在设置页切换五套主题

### 导入题库

首页 →「导入题库」→ 选择 `.docx` 文件 → AI 自动解析 → 预览确认后入库

### 刷题

首页勾选题库 →「定向爆破」→ 选择题目数量 → 选择模式：

- **正常刷题**：答完即判，立刻校对解析
- **练习**：选择限时/不限时 → 答题卡漫游 → 统一提交批改
- **背题模式**：直接高亮答案，不计统计

**题型操作**：
- 单选题：点击选项直接判定
- 多选题：点击选项勾选，底部确认提交
- 判断题：点击「✓ 正确」或「✗ 错误」
- 填空题：逐空输入后提交
- 名解/简答/问答：文本框作答，提交后智能比对

**交互**：
- 左右滑动：切换上/下一题
- 答题后手动点击「查看 AI 解析」

### 练习模式

点首页「练习」→ 选限时/不限时：
- **不限时**：正计时，自由跳题，手动「提交练习」
- **限时**：选 15~120 分钟或自定义，倒计时归零自动交卷
- 答题卡网格视图：圆点分色显示当前/已答/未答

### 背题模式

首页「背题模式」→ 正确选项绿色高亮，不记录统计，底部上一题/下一题自由翻页

### 错题本

首页「错题本」→ FSRS 算法根据遗忘曲线自动排期，只显示当前需要复习的错题

### 调试日志

设置页 → 调试日志 → 可开启 / 导出 / 清空 / 分享
