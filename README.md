# 呆猫刷题宝

DeepSeek AI 驱动的本地刷题助手 · 支持单选/多选/判断/填空/名解/简答/问答 · Windows + Android 双端

本软件由 B站 **笨蛋鱼坏蛋猫** 开发

## 特别说明

本软件只是因为作者受不了市面上的收费会员制的刷题软件，借助 deepcode 开发的及其低质的小软件，因为专业类别不是计算机，后续维护可能无法实现，还请谅解。

## 功能一览

| 功能 | 说明 |
|------|------|
| 📥 DOCX 导入 | 上传 .docx 文件，AI 自动提取题目、选项、答案 |
| 🧠 AI 解析 | 每道题自动生成考点分析、解题思路、答案详解 |
| 💬 AI 追问 | 对解析内容继续提问，AI 深入讲解 |
| 📊 薄弱分析 | 自动统计各题库正确率 |
| 📝 错题本 | FSRS 间隔重复算法，智能排期错题复习 |
| 👁 背题模式 | 一键切换，直接显示答案快速浏览 |
| 🎵 答题音效 | 对错提示音，可在设置中开关 |
| 📋 题型支持 | 单选题、多选题、判断题、填空题、名解、简答、问答 |
| 🌓 暗色模式 | 跟随系统自动切换明暗主题 |
| 📈 刷题小结 | 每轮结束自动展示答题统计 |

## 安装

### Windows

1. 下载 `damo-1.26.6.17-Windows.zip` 解压
2. 双击 `flashcard_app.exe` 运行

### Android

1. 下载 `damo-1.26.6.17.apk` 传到手机
2. 允许安装未知来源应用后安装

## 使用指南

### 配置

右上角设置 → 填入 [DeepSeek API Key](https://platform.deepseek.com/api_keys)

### 导入题库

首页 →「导入题库」→ 选择 `.docx` 文件 → AI 自动解析 → 预览确认后入库

### 刷题

首页「定向爆破」→ 正常刷题

- 单选题：点击选项直接判定，答对 0.6s 自动下一题
- 多选题：点击选项勾选，底部确认提交
- 判断题：点击「✓ 正确」或「✗ 错误」
- 填空题：自动检测空位数量，逐空输入后提交
- 名解/简答/问答：多行文本框作答，提交后智能比对

### 背题模式

刷题界面右上角 👁 图标 → 切换为背题模式，直接展示参考答案和解析，可用上/下题键快速翻阅

### 错题本

首页「错题本」→ FSRS 算法根据遗忘曲线自动排期，只显示当前需要复习的错题

### 调试日志

设置页 → 调试日志 → 可导出 / 清空 / 分享
